# โฟลเดอร์รูปภาพห้องพัก

## วิธีเพิ่มรูปภาพ

1. วางไฟล์รูปภาพในโฟลเดอร์นี้
2. ตั้งชื่อไฟล์ให้ตรงกับชื่อในฐานข้อมูล:
   - `standard.jpg` - ห้อง Standard Room
   - `deluxe.jpg` - ห้อง Deluxe Room
   - `suite.jpg` - ห้อง Suite Room

## รูปภาพที่แนะนำ
- ขนาด: 800x600 พิกเซล
- ไฟล์: JPG, PNG
- ขนาดไม่เกิน: 500KB
